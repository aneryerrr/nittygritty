Extract Reservations
Upload Radixx export → enrich connections → download
Step 1: In Radixx, set DEP DATE FROM & TO → Search → Uncheck Pending & Cancelled → Search again → Export to Excel.
Step 2: Upload the exported .xlsx here.
Step 3: If CONN data was entered in Radixx, it will be extracted automatically. If the CONN/Comments columns are empty (resorts haven't entered data yet), use the Connection Input Panel that appears after upload to manually enter connections per PNR group.
Rules — IATA RSADGL01: read CONN from Address1 field. All others: read from Comments.
Last comment entry per PNR wins. Flight time looked up from active schedule (STA for arrivals, STD for departures). Missing times flagged △.
PTC: ADT→blank   CHD→C   INF→I   |   A/D: Orig=MLE→A(Arrival)   other→D(Departure)
